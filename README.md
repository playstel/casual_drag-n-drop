# drag_n_drop_MVP

Simple drag-n-drop game (Unity 2023.2.17f1)

-Additional instruments: DOTween, Zenject, UniTask

-2 scenes: Boot, Game (but you can run Game scene directly)

-infinity playing without scene reload
